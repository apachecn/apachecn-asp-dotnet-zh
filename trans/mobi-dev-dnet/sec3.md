# 第 3 节:Azure 云服务

创建现代移动应用程序通常需要健壮的服务后端和基础设施。 Azure 云基础设施为开发人员提供了广泛的服务和开发平台，以创建可与移动应用程序一起使用的。net Core 组件。 这些服务从简单的**平台即服务**(**PaaS**)托管组件到复杂的多模型持久性存储。
