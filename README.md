## 下载

### Docker

```
docker pull apachecn0/apachecn-asp-dotnet-zh
docker run -tid -p <port>:80 apachecn0/apachecn-asp-dotnet-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-asp-dotnet-zh
apachecn-asp-dotnet-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```

apachecn-asp-dotnet-zh <port>
# 访问 http://localhost:{port} 查看文档
```
